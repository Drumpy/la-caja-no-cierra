# Content Bible

## Regla de pedidos

Los pedidos deben sonar como compras reales de almacen/kiosco. Las combinaciones raras quedan reservadas para anomalías, clientes imposibles o efectos de ticket.

Cada personaje tiene tres capas:

1. **Pedido humano**: cotidiano, reconocible, con logica social.
2. **Pedido tensionado**: plausible, pero con una pieza rara.
3. **Pedido imposible**: combinacion sobrenatural que activa ticket/libreta/anomalia.

Ejemplo:

```text
Taxista humano: cafe + cigarros ficticios + chicles
Taxista tensionado: cafe + vela
Taxista imposible: cafe + foto carnet quemada + leche vencida
```

## Tabaco y rating

Si hay cigarros:

- son ficticios, sin marcas reales;
- no se exhiben en gondola;
- se piden por ventanilla;
- salen de cajon o lista textual detras del mostrador;
- no se muestran menores comprandolos;
- no se muestra consumo en pantalla para el prototipo.

Motivo: mantenerlo como referencia adulta de kiosco sin convertirlo en mecanica aspiracional ni subir riesgo de rating/percepcion.

## Clientes

### Taxista

Funcion: rumor de ciudad.

Pedido humano:

- cafe + cigarros ficticios;
- cafe + chicles;
- cafe + alfajor;
- cafe + encendedor;
- agua + chicles + cigarros ficticios;
- bizcochos + cafe si es temprano.

Pedido tensionado:

- cafe + vela;
- cigarros ficticios + pilas + radio;
- cafe + boleto viejo;
- encendedor + paraguas mojado cuando no llueve.

Regla:

- pilas no son rutina del taxista; si las pide, algo del taxi/radio/calle fallo.

### Inspector municipal

Funcion: burocracia absurda / amenaza de clausura.

No compra como cliente normal. Pide:

- habilitacion;
- lista de precios;
- libreta;
- comprobante;
- cartel visible;
- limpieza;
- producto oculto.

Pedido imposible:

- habilitacion de 1987;

- comprobante de una venta futura;
- nombre del dueno anterior.

## Simbolos de ticket

- Mate: memoria, costumbre, repeticion.
- Humo: secreto adulto, rutina nocturna, ocultamiento.
- Tambor: proteccion, llamado, ritmo.
- Libreta: deuda, promesa, consecuencia futura.
- Rambla: frontera, niebla, agua.
- Gato: testigo, azar, noche.
- Futbol: euforia, supersticion, violencia contenida.
- Feria: intercambio, objetos usados.
- Lluvia: hambre, frio, tortas fritas, apagon.
- Tablado: mascara, verdad cantada.
- Taxi: rumor, viaje, destino.
- Boleto: transito, desaparicion.
- Vela: proteccion barata.
