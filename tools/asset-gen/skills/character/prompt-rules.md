# Character Animation Rules

- Output a 2D full-body character sprite sheet for a PlayCanvas 2.5D billboard.
- Background must be perfectly flat solid `#00FF00` for chroma-key removal.
- No floor, shadow, scenery, props, UI, speech bubbles, text, watermark, or logo.
- Keep the whole human body inside every frame with padding.
- Keep the same identity, costume, scale, and foot baseline across all frames.
- Use a single horizontal row with the exact requested frame count.
- Human anatomy must be believable: head, torso, arms, legs, hands and feet visible.
- The animation should read at in-game size behind a kiosk window.
- Style: hand-painted game character, cozy horror, readable silhouette, not photorealistic.
